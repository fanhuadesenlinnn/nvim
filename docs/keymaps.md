# 快捷键说明

`<Space>` 是 leader 键。按下 `<Space>` 后稍等一下，`which-key.nvim` 会在 Neovim 或 Neovide 里显示可用快捷键。

## 日常操作

| 快捷键 | 作用 |
| --- | --- |
| `<Space><Space>` | 查找文件 |
| `<Space>w` | 保存当前文件 |
| `<Space>q` | 关闭当前窗口 |
| `<Space>Q` | 退出 Neovim |
| `<Esc>` | 清除搜索高亮 |

## 文件和搜索

| 快捷键 | 作用 |
| --- | --- |
| `<Space>e` | 打开/关闭文件树 |
| `<Space>E` | 在文件树中定位当前文件 |
| `<Space>ff` | 用 Telescope 查找文件 |
| `<Space>fg` | 用 Telescope 全文搜索 |
| `<Space>fb` | 查找已打开的 buffer |
| `<Space>fh` | 查找帮助文档 |
| `<Space>fr` | 查找最近文件 |

文件树内部额外快捷键：

| 快捷键 | 作用 |
| --- | --- |
| `/` | 在文件树中筛选 |
| `<Esc>` | 清除文件树筛选 |

文件树默认的 `f/F` 筛选快捷键已关闭，避免和全局 `f` 跳转冲突。

## 快速跳转

| 快捷键 | 作用 |
| --- | --- |
| `f` | 当前 tab 所有可见窗口内按字符跳转：按 `f`，输入目标字符，再输入标签 |
| `s` | Flash 快速跳转 |
| `S` | 按语法结构跳转 |
| `r` | operator-pending 模式下远程选择操作目标 |
| `R` | 可视/operator 模式下按语法结构搜索 |

## 多光标

| 快捷键 | 作用 |
| --- | --- |
| `<C-n>` | 选择光标下的单词；继续按会选择下一个相同单词 |
| `<C-Down>` | 向下一行增加一个光标 |
| `<C-Up>` | 向上一行增加一个光标 |
| `n` | 多光标模式中跳到下一个匹配 |
| `N` | 多光标模式中跳到上一个匹配 |
| `q` | 跳过当前匹配 |
| `Q` | 删除当前光标/选择 |
| `Tab` | 在多光标的 cursor 模式和 extend 模式之间切换 |
| `i` / `a` / `c` | 开始同时插入、追加或修改多个光标位置 |
| `<Esc>` | 退出多光标模式 |

如果想做列编辑，也可以优先使用 Neovim 自带的可视块模式：按 `<C-v>` 选择矩形区域，再按 `I` 或 `A` 批量插入。

## 窗口和 buffer

| 快捷键 | 作用 |
| --- | --- |
| `<C-h>` | 切到左侧窗口 |
| `<C-j>` | 切到下方窗口 |
| `<C-k>` | 切到上方窗口 |
| `<C-l>` | 切到右侧窗口 |
| `<S-h>` | 上一个 buffer |
| `<S-l>` | 下一个 buffer |
| `<Space>bb` | 选择一个已打开文件 |
| `<Space>bc` | 选择并关闭一个已打开文件 |
| `<Space>bd` | 关闭当前 buffer |
| `<Space>bp` | 固定/取消固定当前 buffer |
| `<Space>bP` | 关闭所有未固定 buffer |
| `<Space>bo` | 关闭其他 buffer |
| `<Space>br` | 关闭右侧 buffer |
| `<Space>bl` | 关闭左侧 buffer |
| `<Space>sv` | 垂直分屏 |
| `<Space>sh` | 水平分屏 |
| `<Space>se` | 平均分配分屏大小 |
| `<Space>sx` | 关闭当前分屏 |

## 撤销历史

| 快捷键 | 作用 |
| --- | --- |
| `<Space>uu` | 打开/关闭撤销历史树 |

普通撤销仍然用 `u`，重做仍然用 `<C-r>`；`<Space>uu` 是打开一个可视化面板，方便查看和跳回历史节点。

## 终端

| 快捷键 | 作用 |
| --- | --- |
| `<Space>tt` | 打开/关闭浮动终端 |
| `<Space>th` | 打开/关闭下方横向终端 |
| `<Space>tv` | 打开/关闭右侧纵向终端 |
| `<Space>tn` | 新建一个浮动终端 |
| `<Space>ts` | 选择已有终端 |
| `<Esc>` | 在终端里退出 terminal 输入模式，回到普通模式 |
| `<C-h>` | 在终端里切到左侧窗口 |
| `<C-j>` | 在终端里切到下方窗口 |
| `<C-k>` | 在终端里切到上方窗口 |
| `<C-l>` | 在终端里切到右侧窗口 |

## Git

| 快捷键 | 作用 |
| --- | --- |
| `]g` | 下一个 Git 改动块 |
| `[g` | 上一个 Git 改动块 |
| `<Space>gs` | 暂存当前 Git 改动块 |
| `<Space>gr` | 撤销当前 Git 改动块 |
| `<Space>gS` | 暂存当前文件全部改动 |
| `<Space>gR` | 撤销当前文件全部改动 |
| `<Space>gu` | 取消暂存当前 Git 改动块 |
| `<Space>gp` | 预览当前 Git 改动块 |
| `<Space>gb` | 查看当前行 Git 责任信息 |
| `<Space>gd` | 查看当前文件 Git 差异 |
| `<Space>gw` | 切换词级 Git 差异 |

可视模式下也可以选中几行后按 `<Space>gs` 或 `<Space>gr`，只暂存或撤销选中的改动范围。
